# Cybersecurity Fundamentals Project

This project documents the core concepts of **Cybersecurity** and **Web Application Security** learned during the study, focusing on real-world understanding and exam readiness.

---

## Topics Covered

- Introduction to Cybersecurity  
- CIA Triad (Confidentiality, Integrity, Availability)  
- Common Attack Surfaces  
  - Web Applications  
  - APIs  
  - Mobile Applications  
  - Networks  
  - Cloud Infrastructure  
- Mapping Daily-Used Applications to Attack Surfaces  
  - Email  
  - Messaging Applications  
  - Banking Applications  
- Data Flow in Applications  
  - User → Application → Server → Database  
- Attack Points in the Data Flow  
- OWASP Top 10 Overview  
- Security Best Practices and Key Takeaways
